# داشبورد عملیات فروشگاه اینترنتی

یک نمونه کار تعاملی برای نمایش توانایی در طراحی پنل مدیریتی فروشگاه اینترنتی، مدیریت سفارش، پایش اتصال ها و کنترل سلامت سرویس های وابسته.

## لینک ها

- نسخه آنلاین: https://hoomko-commerce-ops.vercel.app
- مخزن GitHub: https://github.com/erenhooman31/hoomko-commerce-ops

## قابلیت ها

- داشبورد RTL و فارسی برای سفارش ها و کانال های فروش
- صفحات داخلی کامل برای نمای کلی، سفارش ها، موجودی، اتوماسیون و گزارش ها
- فیلتر سفارش ها بر اساس WooCommerce، Shopify و Magento
- نمایش وضعیت پرداخت، پیامک، موجودی و Shipping API
- طراحی واکنش گرا برای دسکتاپ و موبایل
- پیاده سازی با React و Vite بدون نیاز به بک اند واقعی

## هدف نمونه کار

این پروژه برای نمایش مهارت در توسعه رابط کاربری مدیریتی، فروشگاه اینترنتی، API Integration و طراحی داشبوردهای قابل استفاده برای کسب و کار ساخته شده است.

## اجرای محلی

```bash
npm install
npm run dev
```
